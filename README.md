# Jetpac (ZX Spectrum)

**Jetpac (ZX Spectrum)**  
Project by **@bypauu & 9-Bit Team**  
March 2025  

A representation of the classic **Jetpac** game originally released for the **ZX Spectrum**.

---

## Description

This project is a recreation of the classic *Jetpac*, aiming to replicate its original gameplay, visual style, and core mechanics. The main purpose of the project is educational, serving as a reference for students and developers learning graphics programming in C++.

The code is shared with the intention of helping others, following the same spirit of learning and collaboration that helped the author in the past.

---

## Technologies Used

- **Language:** C++  
- **Graphics Library:** ESAT Library  
- **Target Platform:** PC  

---

## Important Notice

To **compile and run this project, the ESAT graphics library is required**.

> **ESAT is a private library**, so unless you are part of the institution or have official access, you will not be able to compile the game.

This repository is published **for educational purposes**, allowing students to study the code even if they cannot execute it.

---

## Compilation

Compilation is only possible if you have access to the ESAT library.  
If available, compile the project as a standard C++ application, making sure the ESAT library is correctly linked.
